# Images - ENSA Fès Application

## Images à ajouter

### hero-bg.jpg
- **Usage**: Arrière-plan des pages d'accueil et connexion
- **Dimensions recommandées**: 1920x1280 pixels minimum
- **Format**: JPEG (optimisé pour web)
- **Taille maximale**: 500 KB
- **Description**: Photo du bâtiment ENSA Fès

### Comment ajouter l'image

1. **Préparer l'image**:
   - Redimensionner à 1920x1280px
   - Compresser en JPEG (85-90% qualité)
   - Nommer le fichier `hero-bg.jpg`

2. **Placer dans le dossier**:
   - Copier `hero-bg.jpg` dans ce dossier (`assets/images/`)

3. **C'est tout!**
   - L'image s'affichera automatiquement comme arrière-plan

## Outils recommandés

- **Compression**: TinyJPG, ImageOptim
- **Redimensionnement**: GIMP, Photoshop, ou outils online
- **Formats**: JPG (photos), PNG (icons), SVG (logos)
